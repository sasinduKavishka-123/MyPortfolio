Item Management System

This Java-based terminal application serves as a centralized Command Line Interface (CLI) for orchestrating the core logistics of a delivery service. Built using a functional approach, the application focuses on direct data manipulation and efficient memory usage to handle the day-to-day operations like customer,order, item management of a delivery business.

  1. Centralized Record Keeping: Uses consolidated data structures (like arrays or lists) to store and retrieve details for items, suppliers, and users in real-time.
  2. Direct Interaction: A menu-driven interface allows users to perform CRUD (Create, Read, Update, Delete) operations instantly through simple terminal commands.
  3. Modular Methods: Different tasks—like "Add Customer" or "View Inventory"—are handled by dedicated functions to keep the code organized.

Zombie Game

This C#-driven terminal game is a text-based survival simulation where the player’s primary lifeline is their pickup truck. Forced to navigate a collapsing world, you must make high-stakes tactical decisions where every choice affects the truck's condition and your ultimate survival. Moving away from logistics and management, this application focuses on game state logic, player decision-making and randomized encounters within a post-apocalyptic setting.

  1. Console Manipulation: Uses the Console class for UI flourishes like changing text colors (e.g., health, distance, time) and clearing the buffer to simulate "scene" transitions.
  2. Survival Mechanics: Players must balance finite resources—such as health, time, and fuel navigating through zombie-infested zones.
  3. Control Flow: Employs robust switch statements and nested loops to manage the game’s menu system and combat sequences.

Item Management System

A Software Application that helps to manage orders, items, customers and payments for an agricultural spare part business by using JavaFX, MySQL and Jasper Report. By moving away from terminal-based logic and into a Graphical User Interface (GUI), the system provides a professional, high-performance environment for managing high-volume retail and wholesale operations.

  1. Customer & Credit Tracking: Maintains a detailed history of customer interactions, including contact details and outstanding balances for those purchasing parts on credit.
  2. Layered Architecture: The application is structured using a Layered Architecture, which organizes the code into distinct responsibilities
  3. JavaFX Desktop Interface: Provides a modern, responsive UI with interactive tables, buttons, and navigation menus, making it intuitive for warehouse staff and administrators.
  4. MySQL Persistence: All business-critical data is stored in a relational database, allowing for complex queries, multi-user access, and long-term data security.
